DRV_ReadPortWord 

INT32S DRV_ReadPortWord(
   PTR_T DeviceHandle, 
   PT_ReadPortword  *lpReadPortWord );

Purpose

On the device indicated by the DeviceHandle, it reads a word (two bytes) of status from a system I/O port. The port specifies the absolute I/O address, which is valid only within the range of space allocated by the system.

Parameters

Some parameters, such as specified I/O port address (port) and read back status (WordData), are stored in the PT_ReadPortWord structure pointed by the lpReadPortWord.

Name Direction Range Description
DeviceHandle input default The pointer retrieved from the DRV_DeviceOpen which specifies the target device.
lpReadPortWord input / output default The pointer to indicate the user-prepared structure PT_ReadPortWord that stores specified I/O port address (INT16U port) and read back status (INT16U *WordData).

Return

   1. 0 Successful.
   2. InvalidDeviceHandle The DeviceHandle is NULL.
   3. Other value Please reference ErrorCodes.
