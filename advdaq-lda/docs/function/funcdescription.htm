Using the Functions

Functions we provide can be divided into the following groups:

Device Function Group

Analog Input Function Group

Analog Output Function Group

Digital Input/Output Function Group

Port Function Group

Counter Function Group

Temperature Measurement Function Group

Event Function Group

 

For each function group, calling flows are provided as an overview, followed by detailed descriptions of functions and data structures(Refer to About Function Description and About Data Structure to get a basic idea).

Reading calling flows before individual functions is highly recommended.
