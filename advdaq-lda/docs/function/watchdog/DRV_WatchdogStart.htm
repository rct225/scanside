DRV_WatchdogStart

INT32S DRV_WatchdogStart(PTR_T DeviceHandle);

Purpose

Initiates the watchdog.

Parameters

Name

Direction

Range

Description

DeviceHandle

input

default

The pointer retrieved from the DRV_DeviceOpen which specifies the target device.

Return

   1. 0

Successful.

   2. InvalidDeviceHandle

The DeviceHandle is NULL.
