DRV_GetFIFOSize

INT32S DRV_GetFIFOSize(
   PTR_T DeviceHandel,
   INT32U * lSize );

Purpose

On the device indicated by the DeviceHandle, it retrieves the FIFO size.

Parameters

Name Direction Range Description
DeviceHandle input default The pointer retrieved from the DRV_DeviceOpen which specifies the target device.
lSize output default The FIFO size.

Return

   1. 0 Successful.
   2. InvalidDriverHandle The DriverHandle is NULL.

Note
