PT_AIBinaryIn 

typedef struct tagPT_AIBinaryIn
{
    INT16U chan;
    INT16U TrigMode;
    INT16U *reading;
} PT_AIBinaryIn, * LPT_AIBinaryIn;

PT_AIBinaryIn is used in function DRV_AIBinaryIn.

Member Description

Name Direction Range Description
chan input 0-(N-1) (N is the maximum channel number and depends on the hardware.) The target sampling channel.
From a minimum of zero for the first AI channel to its maximum AI channel number minus one. For example, the range is from 0 to (N-1) on a device with N AI channels.
TrigMode input 0: internal
1: external 		The triggering mode.
 
reading output Depends on the hardware. The pointer that indicates the user prepared space for the return raw data.
