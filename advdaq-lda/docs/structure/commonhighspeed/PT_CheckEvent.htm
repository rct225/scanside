PT_CheckEvent 

typedef struct tagPT_CheckEvent
{
    INT32U EventType;
    INT32U Milliseconds;
} PT_CheckEvent, * LPT_CheckEvent;

PT_CheckEvent is used in function DRV_CheckEvent.

Member Description

Name Direction Range Description
EventType output Depends on the event enabled by the DRV_EnableEvent. The type of the currently signaled event.
Milliseconds input From 0 to 232-1 or INFINITE The time-out period in milliseconds.
If no event occurs during this time, a timeout  event type will be returned by the EventType.
