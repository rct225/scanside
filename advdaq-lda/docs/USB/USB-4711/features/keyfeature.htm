Device Features

Analog Input

  • 16-channel Single-Ended A/D Input for USB4711/A,8-channel Differential-Ended A/D Input for USB4711A
  • 12-bit A/D conversion
  • Sampling rate for USB4711 form 1 Hz  to 100 kHz 
  • Sampling rate for USB4711A form 1 Hz  to 150 kHz 
  • Input Range : +/-10, +/-5, +/-2.5, +/-1.25, +/-0.625
  • Automatic Channel/Gain Scanning

Analog Output

  • 2-channel D/A Output  
  • Output Range with internal reference (V):  0~5, 0~10, +/-5, +/-10

Digital Input

  • 8-channel Digital Input 

Digital Output

  • 8-channel Digital Output 

Counter

  • USB4711: one 32 bits counter with max to 1k input rate
  • USB4711A : channel 0 is a 32 bits event counter with max to 1k input rate or 0.1HZ~10KHZ frequency measure. channel 1 is a  24 bits counter with 24M base clock provides pulse out function .

Note:

1. The USB chip on your system may have a limitation on the number of USB devices it supports. Normally, only five USB-4711/A devices can be supported.

 

2. The maximum sampling rate under Full Speed mode (USB 1.1) is 25kS/s.
