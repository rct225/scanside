Device Features

Analog Input

  • 8-channel Single-Ended or 4-channel Differential A/D Input
  • 12-bit A/D conversion(11-bit A/D conversion in Single-Ended)
  • Sampling rate from 32 Hz to 10 kHz
  • Input Range (V): +/-20, +/-10, +/-5, +/-4, +/-2.5, +/-2, +/-1.25, +/-1

Analog Output

  • 2-channel 12-bit D/A Output
  • Output Range with internal reference (V):  0~5

Digital Input

  • 8-channel Digital Input

Digital Output

  • 8-channel Digital Output

Counter

  • channel 0: 32 bits event counter with up to 5M input rate.

    Note:    1.  If the Analog Input using Fast mode transmission, can not run DA converter 

                 2.  If the Analog Input using Fast mode transmission, the other command interval must greater than 1ms
