Device Features

Digital Input

  • 16 isolated digital input channels
  • Wide input range (5 ~ 60 VDC )
  • 2 channel interrupt handling capability
  • High-voltage isolation on input channels (2500Vdc)

Digital Output

  • 16 isolated digital output channels
  • High-voltage isolation for output channels (2,500 VDC )
  • Wide output range (5 ~ 40 VDC )
  • High-sink current for isolated output channels
  • Current protect for each port 
  • Digital output value retained after hot system reset 
  • Programmable Power-Up States for output channels

Counter

  • 2 independent  CHANNEL Event Counting
  • Resolution: 32-bit counters (low 16bit by hardware and high 16bit by firmware) 
  • Maximum INPUT frequency :1 MHz

Others

  • USB 2.0 compatible
