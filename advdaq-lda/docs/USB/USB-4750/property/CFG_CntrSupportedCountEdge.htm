CFG_CntrSupportedCountEdge

CFG_CntrSupportedCountEdge gets the supported trigger edge of the two counter channel. For usb4751, both counter channels only support falling edge.

Description

 

Type

INT32U[]

Length

4 bytes * counter numbers

Access type

Read

Value

CNTR_CountEdge_FallingEdge(0x2).
