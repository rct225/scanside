Device Features

Digital Input

  • 8 isolated digital input channels
  • Digital input Range:5~30 Vdc
  • 8 channel interrupt handling capability
  • High-voltage isolation on input channels (2500Vdc)

Digital Output

  • 8 Form C-type relay output channels
  • Output status read-back
  • LED indicators to show activated relays
  • Programmable Power-Up States

Others

  • USB 2.0 compatible
