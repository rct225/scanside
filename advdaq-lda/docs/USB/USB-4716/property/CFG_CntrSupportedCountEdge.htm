CFG_CntrSupportedCountEdge

CFG_CntrSupportedCountEdge gets the count edge.

Description

 

Type INT32U
Length 4 bytes
Access type Read
Value To retrieve the supported count trigger edge of the counter

0x01 : CNTR_CountEdge_RisingEdge

0x02 : CNTR_CountEdge_FaillingEdge
