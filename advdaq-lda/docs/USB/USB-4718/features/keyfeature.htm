Device Features

Analog Input

  • 8-channel differential A/D and thermo couple Input
  • 8-channel differential current Input
  • 16-bit A/D conversion
  • Sampling rate 10Hz
  • Input Range (V): 0-2.5, 0-1.0, 0-0.500, 0-0.100, 0-0.050, 0-0.015
    Input Range (mA): 0-20, 4-20
  • Thermo couple type: J, K, T, E, R, S, B
  • Onboard CJC

Digital Input

  • 8-channel Digital Input

Digital Output

  • 8-channel Digital Output
