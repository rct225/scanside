Quadrature Counting

These functions perform quadratic counter operations. They are only supported by the PCL-833.

DRV_QCounterConfig function configures the quadratic counter. The DRV_QCounterStart function starts the quadratic counter operation. When it is running, use the DRV_QCounterRead function to repeatedly read the count value. Call DRV_CounterReset to stop the counter operation. In addition, the Device Drivers provide the DRV_QCounterConfigSys function to configure the system clock and time period of the quadratic counter.
