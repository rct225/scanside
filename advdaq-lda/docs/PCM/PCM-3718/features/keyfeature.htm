Device Features

Overall Features

16 single-ended or 8 differential analog inputs, jumper selectable
12-bit A/D converter, up to 100 kHz sampling rate with DMA transfer
Software programmable gain value for each analog input channel
Software selectable input range for each analog input channel
Two 8-bit digital input/output channels, TTL compatible
Flexible triggering options: software trigger, programmable pacer trig-
ger and external pulse trigger
Data transfer by program control, interrupt handler routine or DMA

Range Code Table

Each A/D channel has its own individual input range, controlled by a range code stored in on-board RAM. If user want to change the range code for a given channel, select the channel as the start channel in register BASE + 02H, MUX scan, then Write the range code to bits 0 to 3 of BASE +01H.
