CFG_DiInterruptTriggerOnFallingEdge

CFG_DiInterruptTriggerOnFallingEdge gets or sets DI channels which issue interrupt on FALLING Edge.

Description

 

Type

INT8U Array

Length

DI Port Count * 1 bytes

Access type

Read/Write

Value

One bit for one channel. If bit value is 1, the channel will issue interrupt on falling edge, otherwise issue interrupt on rising edge.
