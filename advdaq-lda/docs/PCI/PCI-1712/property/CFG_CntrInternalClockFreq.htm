CFG_CntrInternalClockFreq

CFG_CntrInternalClockFreq gets or sets the internal clock frequency device used for pulse out.

Description

 

Type INT32U array (when get property), INT32U (when set property)
Length sizeof(INT32U) * n( device supported internal clock when get property), sizeof(INT32U) (when set property)
Access type Read/Write
Value Get the property:

Each element of the array indicates a supported clock frequency.

Set the property :

The internal clock frequency for Frequency measurement and Pulse width measurement function

Frequency measurement

0 - 1.3M Hz ~ 10M Hz

1 - 131K Hz ~ 1.3M Hz

2 - 13K Hz ~ 131K Hz

3 - 0.2 Hz ~ 13K Hz

 

Pulse width measurement area

0 - 0.1us ~ 6553.5us

0 - 6553us ~ 65535us

0 - 65535us ~ 655350us

0 - 655350us ~ 6553500us
