CFG_CntrSupportedGateMode

Purpose

To retrieve the supported gate mode, usually used for event counting.

Data Type

INT32U

Data Size

Dependant on hardware.

Access Type

Get only

Description

The already specified gate modes include:

           0x1: No Gate signal or unprogrammable gate mode.( The gate will be set by software internally)
           0x2: High Level active (external gate signal)
           0x4: Low Level active (external gate signal)

If this property is not supported by the device, then the gate mode cannot be selected.
