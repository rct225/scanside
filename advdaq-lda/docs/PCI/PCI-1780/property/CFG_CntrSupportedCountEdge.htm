CFG_CntrSupportedCountEdge

Purpose

To check the count edge supported by the installed device.

Data Type

INT32U

Data Size

Sizeof (INT32U)

Access Type

Get only

Description

Rising Count – 0x1

Falling Count – 0x2

 

By using this property, one can check whether the count edge is selectable for the device.

If this property is not supported by the device, the count edge cannot be chosen.
