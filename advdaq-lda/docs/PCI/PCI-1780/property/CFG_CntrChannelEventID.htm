CFG_CntrChannelEventID

Purpose

To check the Event ID for the installed device.

Data Type

INT32U Array

Data Size

Same as the counter's channel count.

Access Type

Get only

Description

The Event ID of each channel is indicated by a INT32U array.

"0" indicates there is no event occurred on this channel.

This property is added because the counter cannot generate a uniform event ID. For example, the event ID of PCI-1750/PCI-1751 is ADS_EVT_PORT0 and ADS_EVT_PORT1, while the event ID of PCI-1780 is a different one.
