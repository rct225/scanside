Event Type

PCI-1784/U supports the following events:

Event Type Description
ADS_EVT_INTERRUPT Signaled when a certain number of interrupts occur. The number is specified when applications call DRV_EnableEvent.
ADS_EVT_OVERFLOW_CNT0 Signaled when counter 0 overflow occurs.
ADS_EVT_OVERFLOW_CNT1 Signaled when counter 1 overflow occurs.
ADS_EVT_OVERFLOW_CNT2 Signaled when counter 2 overflow occurs.
ADS_EVT_OVERFLOW_CNT3 Signaled when counter 3 overflow occurs.
ADS_EVT_UNDERFLOW_CNT0 Signaled when counter 0 underflow occurs.
ADS_EVT_UNDERFLOW_CNT1 Signaled when counter 1 underflow occurs.
ADS_EVT_UNDERFLOW_CNT2 Signaled when counter 2 underflow occurs.
ADS_EVT_UNDERFLOW_CNT3 Signaled when counter 3 underflow occurs.
ADS_EVT_INDEX_CNT0 Signaled when counter 0 index input occurs.
ADS_EVT_INDEX_CNT1 Signaled when counter 1 index input occurs.
ADS_EVT_INDEX_CNT2 Signaled when counter 2 index input occurs.
ADS_EVT_INDEX_CNT3 Signaled when counter 3 index input occurs.
ADS_EVT_INDEX_CNT3 Signaled when counter 3 index input occurs.
ADS_EVT_DI_INTERRUPT0 Signaled when DI0 input occurs.
ADS_EVT_DI_INTERRUPT1 Signaled when DI1 input occurs.
ADS_EVT_DI_INTERRUPT2 Signaled when DI2 index input occurs.
ADS_EVT_DI_INTERRUPT3 Signaled when DI3 over compare occurs.
ADS_EVT_OVERCOMPARE_CNT0 Signaled when counter 0 over compare occurs.
ADS_EVT_OVERCOMPARE_CNT1 Signaled when counter 1 over compare occurs.
ADS_EVT_OVERCOMPARE_CNT2 Signaled when counter 2 over compare occurs.
ADS_EVT_OVERCOMPARE_CNT3 Signaled when counter 3 over compare occurs.
ADS_EVT_UNDERCOMPARE_CNT0 Signaled when counter 0 under compare occurs.
ADS_EVT_UNDERCOMPARE_CNT1 Signaled when counter 1 under compare occurs.
ADS_EVT_UNDERCOMPARE_CNT2 Signaled when counter 2 under compare occurs.
ADS_EVT_UNDERCOMPARE_CNT3 Signaled when counter 3 under compare occurs.
ADS_EVT_INTERRUPT_TIMER4 Signaled when counter 4 timer input occurs.
ADS_EVT_TIME_OUT Signaled when time out events occur during the wait interval in DRV_CheckEvent.
ADS_EVT_NO_EVENT Signaled when no events occur during the wait interval in DRV_CheckEvent.
