CFG_InterruptTriggerSource

CFG_InterruptTriggerSource gets or sets DI Interrupt Mode

Description

　

Type INT32U[]
Length 4 bytes
Access type Read/Write
Value

Each element represents one group

Parameter Value:

 

Counter 0 overflow interrupt : CNTR0OVERFLOWENABLE

Counter 1 overflow interrupt : CNTR1OVERFLOWENABLE

Counter 2 overflow interrupt : CNTR2OVERFLOWENABLE

Counter 3 overflow interrupt : CNTR3OVERFLOWENABLE

Counter 0 underflow interrupt : CNTR0UNDERFLOWENABLE

Counter 1 underflow interrupt : CNTR1UNDERFLOWENABLE

Counter 2 underflow interrupt : CNTR2UNDERFLOWENABLE

Counter 3 underflow interrupt : CNTR3UNDERFLOWENABLE

Counter 0 index interrupt : CNTR0INDEXENABLE

Counter 1 index interrupt : CNTR1INDEXENABLE

Counter 2 index interrupt : CNTR2INDEXENABLE

Counter 3 index interrupt : CNTR3INDEXENABLE

Counter 0 digital input interrupt : CNTR0DIENABLE

Counter 1 digital input interrupt : CNTR1DIENABLE

Counter 2 digital input interrupt : CNTR2DIENABLE

Counter 3 digital input interrupt : CNTR3DIENABLE

Counter 0 over compare interrupt : CNTR0OVERCOMPAREENABLE

Counter 1 over compare interrupt : CNTR1OVERCOMPAREENABLE

Counter 2 over compare interrupt : CNTR2OVERCOMPAREENABLE

Counter 3 over compare interrupt : CNTR3OVERCOMPAREENABLE

Counter 0 under compare interrupt : CNTR0UNDERCOMPAREENABLE

Counter 1 under compare interrupt : CNTR1UNDERCOMPAREENABLE

Counter 2 under compare interrupt : CNTR2UNDERCOMPAREENABLE

Counter 3 under compare interrupt : CNTR3UNDERCOMPAREENABLE

Timer pulse Interrupt : TIMERPULSEENABLE

IE Overall interrupt : IEOVERRALLENABLE
