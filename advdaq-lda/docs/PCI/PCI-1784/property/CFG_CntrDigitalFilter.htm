CFG_CntrDigitalFilter

CFG_CntrDigitalFilter gets or sets the counters' digital filter enable state of the device. A digital filter is used to eliminate unexpected signals and noise from the card circuitry.

Description

Type INT32U array
Length 4 bytes for one counter channel
Access type Read/Write
Value Each element is the array represents one counter's digital filter enable state.
0: disable digital filter

1: enable digital filter

The user can set different digital filtering parameters for each input channel in different applications. Refer to the example of the PCI-1760 as shown below:

1. When a digital filter is enabled, the PCI-1760 will sample the signals at the enabled input channel at a 5 ms sampling rate.

2. When a high or low signal is present at a digital input channel whose digital filter function is enabled, the signal will be filtered out as noise unless it lasts for an effective period.
