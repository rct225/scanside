CFG_DiInterruptSupportedChannel

CFG_DiInterruptSupportedChannel gets the DI channels which support interrupt function.

Description

Type INT8U array
Length 1 byte for one port, 1 bit for one channel.
Access type Read
Value Each bit in the byte array represents one channel.
If the bit returns "1", means the corresponding channel supports DI Interrupt function, otherwise it doesn't support.
For example, for a 2-port DI device, the size of the array is 2;  when the returned array is [2,5], which means channel1 in port0, channel0 and channel2 in port1, support this function.
