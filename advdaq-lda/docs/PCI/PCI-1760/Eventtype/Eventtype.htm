Event Type

PCI-1760 supports the following events:

Event Type Description
ADS_EVT_PATTERNMATCH_PORT0 Triggered when the received value from digital input channels matches the pre-set match value.
ADS_EVT_COUNTERMATCH Triggered when the counter value reaches a pre-set match value.
ADS_EVT_COUNTEROVERFLOW Triggered when the counter exceeds its maximum value, 65535.
ADS_EVT_STATUSCHANGE_PORT0 Triggered when any enabled digital input channel changes its state.
