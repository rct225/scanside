Event Type

PCI-1758UDIO supports the following events

Event Type Description
ADS_EVT_DI_INTERRUPT0
~ADS_EVT_DI_INTERRUPT63		 Triggered when channel DI0~DI63 generates a certain number of interrupts. The number is specified when applications call DRV_EnableEvent.
ADS_EVT_WATCHDOG_OVERRUN Triggered when the watch dog timer counts down to 0.
