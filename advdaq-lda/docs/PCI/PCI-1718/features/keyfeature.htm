Device Features

16 single-ended or 8 differential analog inputs, switch selectable

12-bit A/D converter, with up to 100 kHz sampling rate with interrupt transfer and different gain for each channel

Software-selectable gain: 0.5, 1, 2, 4, 8 for PCI1718HDU;0.5, 1, 10, 100, 1000 for PCI1718HGU

Software selectable analog input range: +/-0.625, +/-1.25, +/-2.5, +/-5, +/-10, 0~1.25, 0~2.5, 0~5, 0~10V for PCI1718HDU

+/-0.005, +/-0.01, +/-0.05, +/-0.1, +/-0.5, +/-1, +/-5, +/-10, 0~0.01, 0~0.1, 0~1, 0~10V for PCI1718HGU

On-board 1K words FIFO buffer with software selectable interrupt

16 digital inputs and 16 digital outputs, TTL/DTL compatible

One 12-bits analog output channel

Flexible trigger options: software, programmable pacer and external pulse

Data transfers by program control, interrupt handler routine

Automatic channel/Gain scanning

Keeping output values after system reset

Compatible with PCL-818HD

Board ID support
