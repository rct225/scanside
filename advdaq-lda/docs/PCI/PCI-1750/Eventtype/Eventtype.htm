Event Type

PCI-1750 supports the following events:

Event Type

Description

ADS_EVT_DI_INTERRUPT0

Triggered when Group 0 of the device generates a certain number of interrupts by IDI0. The number is specified when applications call DRV_EnableEvent.

ADS_EVT_DI_INTERRUPT8

Triggered when Group 1 of the device generates a certain number of interrupts by IDI8. The number is specified when applications call DRV_EnableEvent.

ADS_EVT_INTERRUPT_TIMER1

Triggered when Group 0 of the device generates a certain number of interrupts by Timer 1. The number is specified when applications call DRV_EnableEvent.

ADS_EVT_TERMINATE_CNT2

Triggered when Group 1 of the device generates a certain number of interrupts by Counter 2. The number is specified when applications call DRV_EnableEvent.
