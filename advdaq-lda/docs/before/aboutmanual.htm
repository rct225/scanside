About This Manual

This manual contains the information you need to get started with Advantech DAQ Device Linux Drivers. The Device Drivers allow you to easily perform versatile I/O operations through properties, methods and events in programs developed with C/C++ Language. This manual contains descriptions of examples we provide, which can be modified for your own applications. This manual does not show you how to solve every possible programming problem. Specific questions should be directed to Advantech's application engineers.
