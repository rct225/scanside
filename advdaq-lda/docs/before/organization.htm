Organization of This Manual

This user manual is divided into the following sections:

  • Advantech DAQ Device Linux Driver Overview
  • Getting Started with Advantech Device Linux Driver
  • Device Driver Programming Examples
  • Functions Reference 

Advantech DAQ Device Linux Driver Overview

Gives the user a basic idea of installing and programming with Advantech DAQ Device Linux Driver.

Getting Started with Advantech DAQ Device Linux Driver

Gives the user a clear idea of Advantech DAQ Device Linux Driver and a walk-through in creating a simple application step-by-step. 

Device Driver Programming Examples

List out the Examples and descriptions provided for each Das Card. The examples introduce how to use Advantech DAQ Device Linux Drivers in your applications to get the most of Advantech's Data Acquisition and Control cards.

Functions Reference

Function Support Tables

A list of all the functions and data structures that are supported by the 32-bit Device Drivers. In addition, it shows what functions are supported by each Advantech hardware(2 Series: PCI and USB Series).

Using the Functions

Firstly gives an overview of function description and data structure, then provides the detailed information of functions by function groups. Each function group begins with function calling flows to give user some hints, followed by a thorough description of all the functions and corresponding data structures.

Error Codes

Explains the error codes that might be returned when the functions are not executed successfully. Refer to this section for debugging purpose.
